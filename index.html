よくある臨床の悩み

  • 介入後の歩行が安定せず、再現性が低い
  • 背屈制限・足趾過緊張が戻る、説明が抽象的になりがち
  • 偏平足／ハイアーチ／浮指で優先順位が立てられない
  • 足底感覚の鈍さを感じるが再教育の手順が不明
  • 神経・構造・感覚入力を一連の流れで統合できていない
なぜ足かを見る

なぜ「足」なのか（神経OS・感覚の視点）

足は“全身のOS”。床情報を最初に取り込む感覚ゲートであり、運動出力の安全判定を左右する。

感覚入力が狂ったままでは、いくら良い運動を入れても脳は“危険”と判断し出力を変えない。

だからこそ「調整（ノイズ除去）→再教育（地図の再構築）→出力確認（安全の再学習）」が必要。

重要メッセージ

調整 → 再教育 → 出力確認 の順番が成果を左右します。

このセミナーの思想（施術≠ゴール）

  • 施術は再教育を成立させるための前処理
  • 調整 → 再教育 → 出力確認 の順番が重要
  • 刺激＝治療ではない。刺激＝「ここを使っていい」という許可
  • “構造を整える”で終えず、“脳・感覚・運動出力”を再設計する

セミナー内容（章立て）

この章で得られること（先出し要約）
  • 評価→調整→再教育→出力確認の“順番の意味”が腹落ちする
  • 距骨／距腿／リスフラン／脂肪体の介入が“なぜ効くか”を説明できる
  • 偏平足・ハイアーチ・浮指の優先順位設計がその場で組める
第1章｜足はなぜ「全身のOS」なのか【概論×生理学】（40分）

神経OSの視点から、手技の必要性への伏線を提示。なぜ順番が成果を左右するかを整理。

第2章｜解剖学・機能解剖学：足部構造と調整ポイント（45分）
  • 距骨（Talus）：情報と荷重のハブ。調整＝情報ルート開通
  • 距腿関節：背屈制限と代償。調整＝安全域の再設定
  • リスフラン関節（内・外）：縦横アーチの切り替えスイッチ
第3章｜神経学：足底感覚と脳内地図＋施術の意味（50分）
  • 足底・足趾感覚入力：マイスナー／メルケル／ルフィニ／パチニ
  • 脂肪体リリース（ケーラー・踵）：センサー解像度を上げる
  • Sensory Blur：環境・固着による地図のぼやけ。施術はリセット
  • 刺激＝「ここを使っていい」という許可
第4章｜足部の崩れを「代償戦略」として読む（45分）
  • 偏平足：距骨＋内側リスフラン
  • ハイアーチ：踵脂肪体＋距腿関節
  • 浮指：足趾MP＋ケーラー脂肪体
第5章｜全身影響マップ × 説明力（30分）

「まず感じられる状態へ。その上で動きを入れると脳が安全だと判断」

第6章｜実技統合：調整 → 再教育 → 出力確認（70分）
  • Step0 構造リセット：距骨／距腿／リスフラン／足趾／脂肪体
  • Step1 マッピング：足底・足趾感覚の解像度向上
  • Step2 アイソレーション：足趾独立／距骨下関節誘導
  • Step3 インテグレーション：ショートフット→立位・歩行→体幹出力確認
実技のゴール

クライアントと同じ場で体幹出力の即時変化を確認し、再現率を高める“手順”を言語化する。
